    Storage settings

    Select Storage type and configure Settings for persistent data.


    All form fields are required

    Information

    Local (disk)

    This is the default setting.

    Data is stored locally on the 2nd disk, in an LVM partition mounted to /data.

    NFS

    Network File System (NFS) storage provides a multitude of configuration options.

    NFS Mount options are fully listed and described in the nfs(5) man page (external link).

    AoE (ATA-over-Ethernet)

    AoE storage provides block storage over ethernet.

    The AoEDevice is a block device defined by the target server, as described in the aoe-stat(8) man page (external link).

    iSCSI

    iSCSI storage provides a SCSI interface over TCP.

    The iSCSITarget is described in the iscsiadm(8) man page (external link).

    NBD

    NBD provides a network block device by connecting to an NBD server over TCP.
